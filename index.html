Cómo instalar

macOS — pasar el aviso de seguridad
La app aún no está firmada por Apple, así que la primera vez macOS la bloquea. Es segura:
  1. Abre el .dmg y arrastra DANASA Studio a Aplicaciones.
  2. Al abrirla saldrá «no se puede verificar el desarrollador» (o «está dañada»).
  3. Ve a Ajustes del Sistema → Privacidad y seguridad, baja hasta el aviso de «DANASA Studio fue bloqueada» y pulsa Abrir de todas formas.
  4. Si persiste, en Terminal: xattr -dr com.apple.quarantine "/Applications/DANASA Studio.app"
Windows — pasar SmartScreen
La app aún no está firmada, así que Windows muestra una advertencia. Es segura:
  1. Ejecuta el .exe.
  2. Si sale «Windows protegió tu PC», pulsa Más información → Ejecutar de todas formas.
  3. Sigue el instalador.
Después de instalar: actualizaciones automáticas
Una vez instalada, la app se actualiza sola (te avisa con un banner y se reinstala al reiniciar). Solo esta primera vez es manual.
